About

Automation Center for EDI (ACE) gives a hassle free experience of running EDI transactions.
The user of this tool should have access to Unix Environments for the selected regions.
This tool can be used to run more that 15 EDI transactions.

The tool gives a detailed result for regression which can be further exported to Excel sheet.
